🎮 LoGak (로각)

로스트아크 아이템 능력치 조합 계산기 Android 앱
게임 유저의 반복적인 계산 불편을 해결하기 위해 기획부터 배포까지 직접 수행한 개인 사이드 프로젝트

---

📱 프로젝트 소개

**LoGak(로각)**은
MMORPG 로스트아크를 플레이하며 직접 느꼈던 불편함에서 출발한 Android 애플리케이션입니다.

아이템 세팅 과정에서 매번 반복적으로 수행해야 했던
능력치 조합 계산 문제를 해결하기 위해
UI 기반 계산기 형태의 모바일 앱으로 기획·개발되었습니다.

“게임을 하면서 불편했던 점을 직접 해결해보자”는 목표로 시작한 첫 Android 프로젝트입니다.

---

🗓️ 개발 기간

2022.01 ~ 2022.02

---

🧩 기획 배경

	•	아이템 옵션에 따라 능력치를 수기로 계산해야 하는 불편함
	•	게임 내에서 해당 계산 기능 미제공
	•	커뮤니티에서도 동일한 불편을 겪는 유저 다수 존재

이를 해결하기 위해 다음과 같은 방향으로 앱을 기획했습니다.
	•	아이템 선택만으로 결과 확인
	•	반복 계산 제거
	•	모바일 환경에 최적화된 UX 제공

---

✨ 주요 기능

🔹 아이템 능력치 조합 계산
	•	아이템 선택 시 능력치 자동 합산
	•	결과 즉시 반영

🔹 Custom View 구현
	•	입력 변경에 따라 실시간 계산 결과 표시
	•	사용자 조작 흐름에 맞춘 커스텀 UI 구성

🔹 UX 중심 설계
	•	게임 내 제공되지 않는 기능을 사용자 관점에서 재설계
	•	불필요한 입력 제거
	•	직관적인 화면 구성

---

👥 사용자 검증 및 개선

	•	로스트아크 커뮤니티 및 실제 유저 대상 의견 수집
	•	반복 사용 중 발생하는 불편 요소 지속 개선
	•	실사용자 니즈 기반 기능 검증 후 배포

🛠️ 기술 스택

	•	Language: Kotlin
	•	Platform: Android
	•	UI: XML View System
	•	Architecture: MVC → MVVM 일부 적용
	•	Custom View
	•	Gradle
	•	Google Play Console 배포 경험

	•	기획 → 개발 → 배포 → 운영 전 과정 단독 수행

	•	사용자 피드백 기반 개선 사이클 경험
	•	모바일 개발자로서의 진로를 명확히 설정한 계기
